O jogo “Caça Cerejas” foi criado por meio de um trabalho acadêmico, proposto pela disciplina de Pensamento Computacional, do curso de Ciência da Computação, da Atitus Educação. 

O intuito do jogo é alcançar a maior pontuação. Para fazer ponto, é necessário que as cerejas caiam dentro da cesta. Quanto mais cerejas cair, mais pontos você faz. Mas, no momento em que uma cereja cai fora da cesta, você perde o jogo. 

Caça Cerejas é um jogo desenvolvido utilizando Python e Pygame. O objetivo do jogador é controlar uma cesta e capturar o maior número possível de cerejas antes que alguma delas caia no chão.

O jogo possui sistema de pontuação, tela inicial personalizada, tela de boas-vindas, tela de recorde, efeitos sonoros, registro de data e horário, sistema de pausa e mensagem de boas-vindas por voz.

O projeto foi desenvolvido utilizando as seguintes tecnologias e bibliotecas:
-Python 3.14.3
-Pygame
-JSON
-Pyttsx3
